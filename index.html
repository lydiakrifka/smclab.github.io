Sound and Music Computing – UPF Barcelona
About This Blog: Documentation of coursework across seven SMC program courses at Universitat Pompeu Fabra's Music Technology Group. Covering digital signal processing, machine learning for audio, music perception research, and information retrieval systems using Python and computational methods.

Program Overview

Master's program in Sound and Music Computing focusing on the intersection of audio signal processing, machine learning, and music information retrieval.

  • Institution: Universitat Pompeu Fabra, Music Technology Group
  • Supervisor: Prof. Xavier Serra
  • Background: Computer Music, Institute of Sonology, Royal Conservatory Den Haag
  • Focus: DSP, ML for audio, MIR, psychoacoustic research

Current Coursework

  • Audio Signal Processing – STFT analysis, sinusoidal modeling, SMS tools
  • Machine Learning for Audio – CNNs, contrastive learning, genre classification
  • Music Perception – Psychoacoustics, experimental design, statistical analysis
  • Music Information Retrieval – Feature extraction, similarity search, recommendation
  • Research Methodology – Literature reviews, meta-analysis, academic writing

Technical Stack

  • Languages: Python, Bash, LaTeX
  • Audio Libraries: librosa, essentia, sms-tools, soundfile
  • ML Frameworks: PyTorch, scikit-learn, pandas
  • Environment: Arch Linux, Neovim, uv package manager
  • Version Control: Git, GitHub (Arch-style naming conventions)
